Im Rahmen unseres Projekts haben wir eine Website entwickelt, die Studierenden die Planung ihres Studiums erleichtert. Auf der Website können Nutzer ihren Studiengang sowie ihr aktuelles Semester angeben. Basierend auf diesen Angaben werden automatisch die Module ausgegeben, die im jeweiligen Semester belegt werden müssen.

Ziel unseres Projekts war es, den oft komplizierten Überblick über Studienverlaufspläne und Modulhandbücher zu vereinfachen. Durch die automatische Zuordnung der Module erhalten Studierende schnell und übersichtlich die Informationen, die sie für ihre Semesterplanung benötigen.
